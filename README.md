### Jasper Utility Library for Anar Framework

#### Usage Guide

```
		<dependency>
	               <groupId>af.gov.anar.lib</groupId>
	                <artifactId>anar-lib-jasper</artifactId>
                        <version>${project.version}</version>
		</dependency>

```
